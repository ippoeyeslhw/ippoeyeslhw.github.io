mandelbrot fractal golang

28 May 2016, 16:56
golang
golang / mandelbrot / fractal / image

go언어의 package문서를 보다보니 image패키지를 보게 되었다. go언어 익숙해질겸 기존에 몇번 작성해보았던 Mandelbrot Fractal 을 만들어 보기로 마음 먹었다. 픽셀 콘트롤 표준라이브러리의 image패키지를 찾아보다보니 픽셀을 Set하는 것처럼 보이는 부분이 있었다. func (p *RGBA) Set(x, y int, c color.Color) RGBA 구조체에 특정 x,y좌표의 픽셀에 Color를 지정하는것으로 보인다. 그렇다면 몇가지 의문이 생긴다. Jpeg, Png 같은 이미지로 저장을 할수 있는지 Color는 무엇인지 이미지 생성 및 저장 간단하게 이미지를 생성 및 저장해보기로 한다. png 포맷으로 저장하는 방법을 찾아보다가 보니 image 패키지의 서브디렉토리에 Png가 있었다.
