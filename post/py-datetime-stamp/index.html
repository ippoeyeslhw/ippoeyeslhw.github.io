Py Datetime Stamp

2017/08/28

Categories: python Tags: python snippets

파이썬에서 Datetime 을 이용하여 unix timestamp 를 구하고 싶을 경우가 있다. 1970-1-1 을 기준으로 차이값(time delta)를 구한뒤에 총 초를 구하고 밀리세컨 단위로 바꿔주면 된다.

import datetime

def unix_timestamp(y,m,d,h):
    dt = datetime.datetime(y,m,d,h)
    epoch = datetime.datetime(1970,1,1)
    return int((dt-epoch).total_seconds() * 1000)

stmp = unix_timestamp(2017,8,28,13)

print (stmp)

혹은 timestamp 메서드를 이용하되 utc timezone 지정을 해주고 출력을 해본다.

import datetime
def unix_timestamp(y,m,d,h):
    dt = datetime.datetime(y,m,d,h)
    return int(dt.replace(tzinfo=datetime.timezone.utc).timestamp() * 1000)

이렇게 구하면 날짜 지정하기가 쉬워진다.
